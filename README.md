#  Smart Notes Pro App

A simple and interactive **Smart Notes web application** built using **HTML, CSS, and JavaScript**, combining **VibeCoding-generated UI** with **manual feature implementation**.
---

## Features
*  Add new notes
*  Edit existing notes
*  Delete notes
*  Pin important notes
*  Case-insensitive search with highlights
*  Sort notes (Newest / Oldest)
*  Persistent storage using `localStorage`
*  Mock AI features (summary, tags, mood detection)
*  Light / Dark mode toggle
*  Responsive design (works on mobile & desktop)
---
##  Tech Stack
* **HTML5** – Structure
* **CSS3** – Styling & themes
* **JavaScript (Vanilla)** – Logic & interactivity
* **GitHub Pages** – Hosting
---
##  VibeCoding Usage
VibeCoding was used to:
* Generate the base UI layout
* Design responsive note cards
* Apply light and dark theme styles
All application logic and features were implemented manually in JavaScript.

---

##  What Was Built Manually

* Add, edit, delete note functionality
* localStorage integration for data persistence
* Real-time search and highlight
* Pinning logic
* Sorting logic
* Mock AI logic for summary, tags, and mood
* DOM manipulation and event handling
